Why Hoppers?

Instant Generation

Upload your image and get an animated emote back in seconds. No waiting, no queues — everything runs right in your browser session.

Batch Processing

Upload up to 20 images at once and animate them all with the same settings. Download individually or grab them all in one click.

Any Image Format

Works with PNG, JPG, WebP, GIF, BMP, and TIFF. Output as GIF for maximum compatibility or APNG for full-color, artifact-free results.

Full Control

Tweak canvas size, frame count, speed, jump height, and tilt angle. Every parameter has sensible defaults and validated ranges.

Discord Ready

Output is sized and optimized for Discord emotes. Warnings appear if the file exceeds the 256 KB limit so you can adjust before uploading.

Open Source

Fully open-source and self-hostable. Run it locally with Python, deploy with Docker, or use the hosted version — your images never leave the server.

5 Animation Styles

Each animation is tuned for emote-sized sprites. Pick one, tweak the settings, and hit generate.

Hop

Left-right tilt with a vertical bounce. The classic emote wiggle.

Spin

Full 360-degree rotation around the center. Satisfying and hypnotic.

Shake

Rapid horizontal vibration. Like a buzzing notification.

Bounce

Vertical bounce with squash-and-stretch on landing. Cartoonish energy.

Pulse

Rhythmic scale-up and scale-down. A heartbeat effect.
